# Лабораторная работа №1 - Нормализация данных в модель "Снежинка"

## Описание
Трансформация данных из CSV-файлов (10 файлов по 1000 строк) в аналитическую модель "Снежинка".

## Структура
- `init/` - скрипты инициализации (создание таблицы mock_data, загрузка CSV)
- `исходные данные/` - CSV файлы с исходными данными
- `ddl.sql` - создание схемы снежинка (измерения + факты)
- `dml.sql` - заполнение схемы из mock_data
- `docker-compose.yml` - запуск PostgreSQL

## Запуск
```bash
docker-compose up -d
