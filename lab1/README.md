# Лабораторная работа №1 - Нормализация данных в модель "Снежинка"

## Описание
Трансформация данных о продажах зоотоваров из CSV-файлов в аналитическую модель "Снежинка".

## Структура репозитория
- `scripts/` - скрипты инициализации (создание staging-таблицы, загрузка CSV)
- `source_data/` - исходные CSV файлы с данными
- `schema.sql` - создание целевой схемы (измерения + факты)
- `data_load.sql` - заполнение целевых таблиц из staging
- `docker-compose.yml` - конфигурация PostgreSQL
